# client

React Vite starter with Tailwind CSS generated with Forge Stack.

## Run

```bash
cd client
npm install
npm run dev
```

## Structure

- `src/App.jsx` for the main app shell
- `src/style.css` for global styles
- `vite.config.js` for Vite configuration

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`

Language: js
